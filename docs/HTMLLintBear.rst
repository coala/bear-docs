`HTMLLintBear <https://github.com/coala/coala-bears/tree/master/bears/upload/HTMLLintBear/coalaHTMLLintBear/HTMLLintBear.py>`_
==============================================================================================================================

Check HTML source code for invalid or misformatted code.

See also <https://pypi.python.org/pypi/html-linter>.

`Supported Languages <../README.rst>`_
--------------------------------------

* HTML

Settings
--------

+----------------------+-----------------------------------------------------------+
| Setting              |  Meaning                                                  |
+======================+===========================================================+
|                      |                                                           |
| ``htmllint_ignore``  | List of checkers to ignore. (Optional, defaults to '()'.) +
|                      |                                                           |
+----------------------+-----------------------------------------------------------+


Dependencies
------------

* ``pip`` - ``html-linter``


Can Detect
----------

* Formatting
* Syntax
